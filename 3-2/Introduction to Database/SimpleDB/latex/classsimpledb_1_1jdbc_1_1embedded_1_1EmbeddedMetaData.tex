Returns the number of characters required to display the specified column. For a string-\/type field, the method simply looks up the field\textquotesingle{}s length in the schema and returns that. For an int-\/type field, the method needs to decide how large integers can be. Here, the method arbitrarily chooses 6 characters, which means that integers over 999,999 will probably get displayed improperly. \mbox{\Hypertarget{classsimpledb_1_1jdbc_1_1embedded_1_1EmbeddedMetaData_a090cf18d364fe42ec078e85745ecd270}\label{classsimpledb_1_1jdbc_1_1embedded_1_1EmbeddedMetaData_a090cf18d364fe42ec078e85745ecd270}}
