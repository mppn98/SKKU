◆ next()

boolean simpledb.materialize.GroupByScan.next ( )
inline

Move to the next group. The key of the group is determined by the group values at the current record. The method repeatedly reads underlying records until it encounters a record having a different key. The aggregation functions are called for each record in the group. The values of the grouping fields for the group are saved.
